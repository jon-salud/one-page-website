Welcome to My UI Testing Page

About

A simple one-page website designed to test various UI behaviors for demoing UI Functional automation using Playwright. It includes elements such as search fields with delayed dropdowns, calendar fields, login fields with modal notifications, date and time fields, and a paginated table.
